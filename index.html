Scope 1 & 2 Emissions Calculator

The Carbon Footprint (Data Input & Compiler) application is designed to help users collect, analyze, and report carbon-related data. It allows companies to input carbon emissions data, calculate emissions across various scopes, and generate comprehensive reports in DOCX format.

This software includes three main components:

Data Collection UI: A user-friendly interface to input various carbon-related data.
Compiler UI: A tool to compile the input data and generate a DOCX report.
Emissions Analysis: A function to calculate the carbon footprint using the data and apply corresponding GHG conversion factors.

The system can be used through executable files (preferred for end-users) or Python scripts (ideal for developers and contributors).

DVLA Reg Analyser

The DVLA Vehicle Data Fetcher is a Python-based GUI application that fetches vehicle details from the DVLA API based on a list of vehicle registration numbers provided in a CSV or Excel file. The program processes each registration number and retrieves details such as engine size, fuel type, and revenue weight, while also categorizing the vehicle's carbon factor.

News Scraper

This application allows users to search for news articles based on predefined client names or custom user queries. It displays the results in a table and allows the user to filter the results and open links directly from the table. The application also supports saving custom search queries between sessions.
